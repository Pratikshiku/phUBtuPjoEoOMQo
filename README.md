## Sports Product Recommendation System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/30116febfb334ecaae075f945c3553cf/ghb20250917) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript` etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): Regular User, Administrator.

**Regular User Functions:**
1. Login
2. Register
3. Homepage
4. Product Information
5. Communication Forum
6. Announcement Information
7. Message Board
8. Background Management
9. Shopping Cart
10. Personal Center
11. My Posts
12. My Orders
13. My Address
14. My Favorites


**Administrator Functions:**
1. Login
2. Personal Center (Change Password, Personal Information)
3. User Management
4. Product Category Management
5. Product Information Management
6. Communication Forum
7. Message Board
8. System Management (Carousel Management, About Us, Announcement Information, System Introduction)
9. Order Management
